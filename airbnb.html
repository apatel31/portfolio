Airbnb's in New York City

This dashboard visualizes Airbnb's in New York City, the pricing by location sheet organizes Airbnb's by zip code and labels the quantity of Airbnbs in each. In addition, the color code helps us identify which zip codes are the priciest. Airbnb's can be filtered out from the dashboard using the tools on the right. We can select which neighborhoods we want to include and exclude in our analysis or we can filter by price or ratings.

The "Bump FOM" sheet looks at rankings of neighborhoods over time. With this visualization, we can see how Airbnb's in New York neighborhoods stack up against each other and how those comparisons have changed over time. "FOM" stands for "Figure of Merit", this sort of equation allows us to apply quantitative value to qualitative data. For example, we can weight how we value Airbnb's by property type.

Finally, the last sheet covers Airbnb prices in New York by property type. With the combination of these sheets and filters, Airbnb users can find the best location and property to rent over Airbnb in New York City.

The dashboard below visualizes a unique scatterplot that graphs the relationship between Airbnb price and ratings by neighborhood in New York City. The scatterplot also demonstrates a timeline in which the user of this dashboard can flip through year by year and see how each neighborhood has changed over time.
