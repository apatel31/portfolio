Football Stats Visualization

This page has a series of visualizations on the top NFL quarterbacks in recent years. The first dashboard is an interactive visual that plots pass completions over time and what wide receivers each quarterback likes to pass to the most. Below that, we can convert the line graph into a bump chart that tracks the ranking of the same quarterbacks by pass completions over time. Finally, the last visual demonstrates the touchdown to interception ratio of each of these quarterbacks.
